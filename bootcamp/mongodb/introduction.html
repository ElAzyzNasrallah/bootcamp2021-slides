La méthodologie MERISE

La méthode Merise est une méthode d'analyse, de conception et de réalisation de systèmes d'informations.

Les Modèles Merise Les plus connus

  • MCD : Modèle Conceptuel de Données
  • MLD : Modèle Logique de Données
  • MCT : Modèle Conceptuel de Traitements
  • MOT : Modèle Organisationnel de Données

MCD

Le MCD est une représentation simplifiée du système à informatiser : Représentation de l'information d'une manière compréhensible par tout le monde.

MCD

Dans un MCD le système à informatiser est perçu comme un ensemble d'objets en relation : Description statique du système d'informations à l'aide du modèle Entité-Association

Les éléments du MCD

  • Les entités
  • Les associations
  • Les attributs
  • Les cardinalités

Les entités

Une entité est un objet (matériel ou immatériel) ayant une existance propre et utile au SI étudié.

Les associations

Une association est un lien entre deux ou plusieurs entités.

Les attributs

Ce sont les informations communes d'une entité ou d'une association. Ces attributs doivent être représentés comme une liste de mots.

Les attributs (suite)

Identifiant : C'est un ou plusieurs attributs permettant d'identifier d'une manière unique un individu de l'entité parmi tous les autres.

Il est généralement souligné pour pouvoir le distinguer.

Les cardinalités

une cardinalité est la combinaison du minimum et du maximum de fois qu'un individu de l'entité est concerné par l'association.
