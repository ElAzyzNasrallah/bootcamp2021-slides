Les formulaires HTML

Un formulaire HTML est utilisé pour collecter des informations de la part de l'utilisateur. Ces dernières sont dans la plupart des cas envoyées à un serveur en vue d'être traitées.

L'élément <form>

Est utilisé pour créer un formulaire.

L'élément <form> est une enveloppe des différents contrôles utilisés dans le formulaire, tels que zones de texte, cases à cocher, boutons radio, listes de choix ...

Principaux attributs de la balise <form>

  • name : définit le nom du formulaire.
  • id : définit l'dentifiant.
  • action : définit l'action à entreprendre quand le formulaire est soumis.
  • method : définit la méthode d'envoi des données du formulaire (GET/POST).
  • autocomplete : définit si le formulaire est doté ou non de la fonctionalité Autocomplete.

Principaux éléments des formulaires

  • input : L'élément de formulaire le plus utilisé. Affiché sous plusieurs formes dépendant de l'attribut type.
  • label : Définit un libellé d'un contrôle du formulaire.
  • select : Définit une liste déroulante.
  • textarea : Définit une zone de texte multi-lignes.
  • button : Définit un bouton dans le formulaire.

Principaux éléments des formulaires (Suite)

  • fieldset : Pour grouper des données ayant une relation sémantique.
  • legend : Définit un titre pour un fieldset.
  • datalist : Définit une liste de valeurs prédéfinies pour un input.
  • option : Définit une option dans une liste
  • optgroup : Définit des groupes d'options dans une liste.
